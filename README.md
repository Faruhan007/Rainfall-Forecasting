# 🌧️ Rainfall Forecasting using Time-Series Analysis

## 📌 Overview
This project focuses on analyzing and forecasting rainfall patterns using historical time-series data.

## 📊 Dataset
- 121 years of rainfall data  
- Monthly observations  

## ⚙️ Approach
- Time-Series Data Analysis  
- Feature Engineering (lag features, moving averages)  
- Model training with proper time-based validation (no shuffling)  

## 📈 Results
- R² Score: **0.6868**  
- RMSE: **181.79 mm**  

## 🔍 Key Insights
- Peak rainfall occurs in July (~757 mm)  
- Seasonal patterns identified in rainfall trends  

## 🛠️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-Learn  

## 🚀 Outcome
Generated forecasts useful for agriculture planning and climate-based decision-making.
